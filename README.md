# AB Testing

A collection of A/B Testing and Bayesian experiments, including Expedia hotel recommendation analysis.

## Contents

- `0_Experimento_A.ipynb` – Experiment A
- `1_Experimento_B.ipynb` – Experiment B
- `2_Expedia_EDA.ipynb` – Expedia Exploratory Data Analysis
- `3_Expedia_Feature_Engineering.ipynb` – Expedia Feature Engineering
- `Bayesian Approach/` – Bootstrapping, fractional stages, and geothermal notebooks
  - `4_Boostraping.ipynb`
  - `frac_stages.ipynb`
  - `geothermal.ipynb`
- `expedia-hotel-recommendations/` – Dataset folder (CSV files excluded from repo due to size)

## Notes

The Expedia dataset CSV files (`train.csv`, `test.csv`, `destinations.csv`, `sample_submission.csv`) are excluded from this repository due to their large size. They can be downloaded from [Kaggle – Expedia Hotel Recommendations](https://www.kaggle.com/c/expedia-hotel-recommendations/data).
